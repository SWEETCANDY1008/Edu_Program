# 값과 표현식

## 기본설명

* 값: 숫자, 문자열, 리스트 등 프로그램이 다루는 기본 대상 
* 표현식: 값, 변수, 함수의 조합으로 생성된 대상
* 표현식 예제
```python
2
"Hello World!"
x
x/5 + f(x, y) + 2**3
```

# 문장과 명령문

## 기본설명

* 문장: 컴퓨터에 어떤 효과를 발생시키는 코딩의 기본 단위.
  * 문장 예제
    * 변수 순언
```
n = 17
```
* 출력문
```python
print(n)
```

* 명령문: 문장을 `if`, `for`, `while` 등과 함께 함께 규칙에 맞게 이어서 만든 결과물
  * 명령문 예제
```python
n = 17
x = n + 3
if x < 30:
    x = x + 3
else:
    print(x) 
```
